# StudyHere_BackEnd

## Project Description
Express.js backend API for StudyHere. Provides RESTful endpoints for lesson management, order processing, and search functionality with MongoDB data storage.

## Live API
**AWS/Render Deployment:** https://studyhere-backend-1-1vqv.onrender.com/endpoint/lessons

## Technology Stack
- **Runtime:** Node.js
- **Framework:** Express.js
- **Database:** MongoDB Atlas
- **Database Driver:** Native MongoDB Node.js Driver
- **Middleware:** CORS, Express static, custom logger
- **Deployment:** Render.com
- **Testing:** Postman

## Features Implemented
-  RESTful API endpoints
-  MongoDB integration with native driver
-  Custom logger middleware
-  Static file serving for images
-  Full-text search functionality
-  CORS enabled for frontend communication
-  Error handling and validation
